# 🌱 Semeador — aplicativo do voluntário

Aplicativo do **voluntário** do projeto Seja Semente: avisos, escalas com
confirmação de presença, agenda (ver e agendar) e perfil. Conversa em tempo
real com a Central (programa Windows e app Seja Semente) através do
Firebase — veja [`../PONTE.md`](../PONTE.md) e o [`../README.md`](../README.md)
do projeto para rodar e configurar.
