Política de Privacidade — Lab Special e Special Clinic

Última atualização: 14 de julho de 2026

Os aplicativos Lab Special (equipe do laboratório) e Special Clinic (dentistas parceiros) formam o sistema de gestão de casos do Laboratório Special (laboratório de prótese dentária). O acesso é exclusivo para a equipe do laboratório e para os dentistas parceiros autorizados — não há criação de conta pelo público: as contas são liberadas pelo gestor do laboratório.

1. Dados coletados

  • Conta Google (e-mail e nome): usada apenas para identificar quem acessa o aplicativo. O acesso é restrito a e-mails previamente autorizados pelo gestor do laboratório.
  • Dados de trabalhos protéticos: nome do paciente, dentista responsável, tipo de trabalho, prazos, etapas de produção, valores e observações — registrados pela própria equipe e pelos dentistas para gestão dos casos.
  • Fotos, vídeos e arquivos: anexados voluntariamente aos trabalhos pela equipe ou pelos dentistas.

2. Como os dados são usados

Os dados são usados exclusivamente para a operação do laboratório: acompanhamento da produção, comunicação com os dentistas, agenda de entregas e controle financeiro interno. Não vendemos, alugamos nem compartilhamos dados com terceiros para fins de publicidade.

3. Armazenamento e segurança

Os dados são armazenados no Google Firebase (Firestore e Authentication), com acesso protegido por login Google e regras de segurança que limitam cada usuário aos dados que lhe dizem respeito. A comunicação é criptografada (HTTPS).

4. Permissões do aplicativo

  • Câmera / arquivos: usada apenas quando o usuário escolhe tirar foto, gravar vídeo ou anexar arquivo a um trabalho.
  • Internet: necessária para sincronizar os dados com o servidor.

5. Exclusão de dados

Para solicitar a exclusão dos seus dados (conta de acesso ou registros associados), entre em contato com o gestor do laboratório pelo e-mail abaixo. Os dados serão removidos em até 30 dias.
